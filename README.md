## Setup
- Add `nuxt-config` dependency using yarn or npm to your project
- Add `nuxt-config` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-config',

    // With options
    ['nuxt-config', { /* module options */ }],
 ]
}
```
